🆓 L3 — Free Style

Design and implement a Shape class hierarchy. Base class: Shape with area() and perimeter(). Subclasses: Circle, Rectangle, Triangle. Each overrides both methods. Demonstrate polymorphism with a list of shapes.

Answer
import math class Shape: def area(self): raise NotImplementedError def perimeter(self): raise NotImplementedError def __str__(self): return f"{type(self).__name__}: area={self.area()}, perimeter={self.perimeter()}" class Circle(Shape): def __init__(self, radius): self.radius = radius def area(self): return round(math.pi * self.radius ** 2, 2) def perimeter(self): return round(2 * math.pi * self.radius, 2) class Rectangle(Shape): def __init__(self, w, h): self.w, self.h = w, h def area(self): return self.w * self.h def perimeter(self): return 2 * (self.w + self.h) class Triangle(Shape): def __init__(self, a, b, c): self.a, self.b, self.c = a, b, c def area(self): s = (self.a + self.b + self.c) / 2 return round(math.sqrt(s*(s-self.a)*(s-self.b)*(s-self.c)), 2) def perimeter(self): return self.a + self.b + self.c for s in [Circle(5), Rectangle(4, 6), Triangle(3, 4, 5)]: print(s)
🎯 L4 — Scenario
Situation
Design a school management system. You need a base class Person (name, age), and two subclasses: Student (adds grade, subject list) and Teacher (adds subject taught, salary). Both have a get_info() method showing different details.
  • Create the Person, Student, and Teacher classes with full encapsulation
  • Use getters/setters for all private attributes
  • Override get_info() in each subclass (polymorphism)
  • Create a list of 3 students and 2 teachers; call get_info() on each
  • Add a comment identifying: inheritance, encapsulation, and polymorphism in your code
Answer
class Person: def __init__(self, name, age): self.__name = name self.__age = age def get_name(self): return self.__name def get_age(self): return self.__age def get_info(self): return f"Name: {self.__name}, Age: {self.__age}" class Student(Person): def __init__(self, name, age, grade, subjects): super().__init__(name, age) self.__grade = grade self.__subjects = subjects def get_info(self): return f"[Student] {super().get_info()}, Grade: {self.__grade}, Subjects: {self.__subjects}" class Teacher(Person): def __init__(self, name, age, subject, salary): super().__init__(name, age) self.__subject = subject self.__salary = salary def get_info(self): return f"[Teacher] {super().get_info()}, Teaches: {self.__subject}, Salary: ${self.__salary}" people = [ Student("Alice", 15, "10A", ["Math", "CS"]), Student("Bob", 14, "9B", ["English", "Art"]), Teacher("Mr. Smith", 40, "Computer Science", 45000), ] for p in people: print(p.get_info()) # Inheritance: Student/Teacher inherit from Person # Encapsulation: all attrs are private (__name, etc.) # Polymorphism: get_info() differs per class
