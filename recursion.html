Recursion — Write & Trace Recursive Algorithms
①
🔁 What is Recursion?
A recursive function calls itself until a base case stops it. Every recursive algorithm needs two things: a base case (stops the recursion) and a recursive case (calls itself with a smaller problem).
Base Case
The stopping condition. Without it, recursion runs forever (stack overflow).
Recursive Case
The function calls itself with a simpler version of the same problem.
🪆 Matryoshka doll analogy
Recursion is like opening a Russian doll. You keep opening smaller dolls until you reach the smallest one — that's the base case. Then you close them back up in reverse order.
②
🐍 Python Examples — Factorial & Fibonacci
Python — recursion.py
# ─── FACTORIAL ────────────────────────── def factorial(n): if n == 0 or n == 1: # base case return 1 return n * factorial(n - 1) # recursive case print(factorial(5)) # 5*4*3*2*1 = 120 # ─── FIBONACCI ────────────────────────── def fibonacci(n): if n <= 1: # base case return n return fibonacci(n-1) + fibonacci(n-2) print(fibonacci(7)) # Output: 13

Fill in the blanks
def recursive_sum(lst): """Recursively sum all elements in a list.""" if ___ == 0: # base case: empty list return ___ return lst[0] + recursive_sum(___) # recursive case print(recursive_sum([1, 2, 3, 4, 5])) # should print 15
Answer
if len(lst) == 0: return 0 return lst[0] + recursive_sum(lst[1:])
🆓 L3 — Free Style

Write a recursive function power(base, exp) that calculates base^exp without using **. Then write countdown(n) that prints n, n-1, ... 1, "Go!" recursively.

💡 Hint: power(base, exp) = base * power(base, exp-1). Base case: exp == 0 returns 1.
Answer
def power(base, exp): if exp == 0: # base case: x^0 = 1 return 1 return base * power(base, exp - 1) # recursive case def countdown(n): if n == 0: # base case print("Go!") return print(n) countdown(n - 1) # recursive case print(power(2, 8)) # 256 print(power(3, 4)) # 81 countdown(5) # 5, 4, 3, 2, 1, Go!

📝 Key Terms
base case
stopping condition
The condition that ends recursion. MUST exist.
recursive case
self-call
The function calls itself with a smaller input
call stack
stack frame
Each call is placed on the stack; unwound when base case is reached
stack overflow
infinite recursion
No base case → too many calls → crash
